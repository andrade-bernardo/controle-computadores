# Controle de Computadores

Sistema simples em Python com interface gráfica para controlar com quem e onde estão os computadores.

## Funcionalidades

- Cadastro de computadores
- Controle por patrimônio
- Marca, modelo e serial
- Responsável pelo equipamento
- Setor
- Localização
- Status do computador
- Observações
- Histórico de alterações
- Banco de dados local SQLite

## Como executar

```bash
python controle_computadores.py
